TEST PROJECT: DOBOT

INTRODUCTION

This is a test website I created based on a Photoshop mockup for a company in Boston, MA. Completed February 2015.

Here are the parameters I was given:

  1. Test on up-to-date Chrome and Firefox down to 320px wide
  2. Don't use any frameworks
  3. Write your code using BEM and SMACSS guidelines
  4. When in mobile view, the content order should be: Heading, dimensions, image, long description, link, etc.

MOCKUP AND DESIGN

Below is the Photoshop mockup of the desktop version of the website. As mentioned in the parameters, the content will display differently in mobile view. There will be one column instead of two, and the images in the right column will be sandwiched between the heading / dimensions and the long description in each row.
